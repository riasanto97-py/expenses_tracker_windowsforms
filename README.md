# Expenses Tracker - Εφαρμογή Windows Forms

## Επισκόπηση

Το Expenses Tracker είναι μια εφαρμογή Windows Forms σε C# που επιτρέπει στον χρήστη να παρακολουθεί, να διαχειρίζεται και να αναλύει τα προσωπικά του έξοδα.

Η εφαρμογή παρέχει λειτουργίες για προσθήκη, επεξεργασία και διαγραφή εξόδων, διαχείριση κατηγοριών, έλεγχο budget και οπτικοποίηση δεδομένων μέσω γραφημάτων και στατιστικών.

---

## Λειτουργίες

### Διαχείριση εξόδων
- Προσθήκη νέου εξόδου με τίτλο, ποσό, κατηγορία και ημερομηνία
- Επεξεργασία υπάρχοντος εξόδου
- Διαγραφή εξόδων
- Προβολή όλων των εξόδων σε DataGridView

### Διαχείριση κατηγοριών
- Προκαθορισμένες κατηγορίες (Food, Transport, Bills, Entertainment, Shopping, Health)
- Δυνατότητα προσθήκης νέων κατηγοριών

### Διαχείριση Budget
- Ορισμός αρχικού budget
- Προσθήκη επιπλέον χρημάτων
- Αυτόματος υπολογισμός:
  - Συνολικών εξόδων
  - Υπολοίπου
- Ειδοποιήσεις budget:
  - Προειδοποίηση όταν το υπόλοιπο πέσει κάτω από 10%
  - Ειδοποίηση όταν ξεπεραστεί το budget

### Στατιστικά & Ανάλυση
- Υπολογισμός μηνιαίων εξόδων
- Εύρεση της κατηγορίας με τα περισσότερα έξοδα
- Οπτικοποίηση εξόδων ανά κατηγορία με γράφημα

### Ρυθμίσεις περιβάλλοντος
- Υποστήριξη θεμάτων (Light, Dark, Gray)
- Επιλογή νομίσματος (€ / $ / £)

## Δομή Project

- `Form1.cs`  
  Περιέχει τη βασική λογική της εφαρμογής και τον χειρισμό του UI.

- `Expense.cs`  
  Μοντέλο δεδομένων που αναπαριστά ένα έξοδο.

---

## Μοντέλο Expense

Η κλάση `Expense` αναπαριστά μία εγγραφή εξόδου στην εφαρμογή.

Περιέχει τις εξής ιδιότητες:

- Τίτλος: Το όνομα ή η περιγραφή του εξόδου
- Ποσό: Η χρηματική αξία του εξόδου
- Κατηγορία: Η κατηγορία στην οποία ανήκει
- Ημερομηνία: Η ημερομηνία καταγραφής

Η κλάση χρησιμοποιείται αποκλειστικά ως δομή δεδομένων και δεν περιέχει λογική.

---

## Εκτέλεση της εφαρμογής

1. Άνοιξε το project στο Visual Studio
2. Κάνε restore τα NuGet packages (αν χρειάζεται)
3. Κάνε build τη λύση
4. Εκτέλεσε την εφαρμογή (F5)

---

## Σημειώσεις

- Όλα τα δεδομένα αποθηκεύονται προσωρινά στη μνήμη (List<Expense>)
- Δεν χρησιμοποιείται βάση δεδομένων
- Τα δεδομένα χάνονται όταν κλείσει η εφαρμογή
- Η εφαρμογή έχει σχεδιαστεί για Windows desktop περιβάλλον
